# Mugshot - Serve PL/Muggin templates over HTTP

Mugshot is an HTTP server that can serve templates over HTTP.
Templates are routed automatically by their metadata.

## Quick start

1. Install plmuggin extension to your PostgreSQL
1. Run example/todomvc.sql
1. Compile (`make`)  and start (`mugshot example/config`)
1. Open http://localhost:3000/all
